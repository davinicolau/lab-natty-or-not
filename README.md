# Nomadismo Digital: Liberdade Geográfica com IA

## Descrição
Este projeto consiste na criação de um ebook educativo com informações práticas sobre o nomadismo digital. O objetivo é orientar pessoas que desejam trabalhar remotamente enquanto viajam pelo mundo, apresentando conceitos, tipos de trabalho, planejamento financeiro, vistos, destinos populares, desafios e checklist para iniciar.

O conteúdo foi produzido com apoio de Inteligência Artificial, demonstrando como ferramentas modernas podem acelerar pesquisa, estruturação e produção editorial.

## Tecnologias Utilizadas
- ChatGPT (conteúdo textual)
- Google Gemini (capa)
- GitHub (repositório e documentação)

## Processo de Criação
1. Definição do tema e do público-alvo.
2. Estruturação dos capítulos com auxílio de IA.
3. Criação da capa do ebook usando IA.
4. Compilação do conteúdo em PDF.
5. Publicação dos arquivos neste repositório.

## Resultados
- Ebook completo em PDF disponível para leitura.
- Capa ilustrativa criada por IA.
- Arquivos públicos e versionados no GitHub.
